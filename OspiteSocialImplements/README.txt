Prerequisiti:

Chiamare in phpmyadmin un database con il nome: "ospitesocial"
ed importare il file ospitesocial.sql

Esecuzuione:

1) Eseguire da cmd il file jar a disposizione:
   java -jar OspiteSocial.jar

2) Caricare la cartella del progetto "OspiteSocial" su di un IDE, ed includere le
   librerie necessarie al funzionamento, queste si trovano nella cartella libraries.
